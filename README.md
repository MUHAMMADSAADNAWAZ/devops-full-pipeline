# DevOps Full Pipeline Project

A minimal Node.js application used to practice real-world DevOps workflows including:

- Docker
- CI/CD
- Kubernetes
- Cloud deployment

## Application
Simple Express API with:
- Root endpoint
- Health check endpoint
- Environment-based configuration

## Endpoints
- `/` – Application info
- `/health` – Health check (used for containers & K8s)

## Run Locally
```bash
npm install
npm run dev
